# FreeCodeCamp Learning Journey

This repository serves as a personal archive of my practical exercises from the freeCodeCamp curriculum. I am using this space to track my daily progress, build consistency, and solidify my foundational knowledge across HTML, CSS, and JavaScript. The project is organized into dedicated folders for each technology.

## JavaScript Exercises

Below is a breakdown of the exercises contained in the `javascript-exercises` folder and the core concepts applied in each.

### 1. `1-greeting-bot`
* **Concepts applied:** Basic variable declaration, reassignment, and string concatenation using the `+` operator.

### 2. `2-bot-facts`
* **Concepts applied:** Dynamically updating sentence outputs by reassigning new values to existing variables.

### 3. `3-sentence-maker`
* **Concepts applied:** Building complex, Mad Libs-style sentences by reusing and updating multiple variables.

### 4. `4-string-explorer`
* **Concepts applied:** Exploring string properties and methods, including template literals, length, character indexing, and `.indexOf()`.

### 5. `5-string-inspector`
* **Concepts applied:** Extracting and inspecting substrings using the case-sensitive `.includes()` and `.slice()` methods.

### 6. `6-string-formatter`
* **Concepts applied:** Formatting strings by removing whitespace, changing text casing, and combining methods to create `camelCase` text.

### 7. `7-string-transformer`
* **Concepts applied:** Transforming strings by swapping specific words using `.replace()` and `.replaceAll()`, and duplicating text using the `.repeat()` method.

### 8. `8-logic-checker`
* **Concepts applied:** Implementing conditional logic and controlling program flow using boolean variables, comparison operators, and `if`/`else` statements.

### 9. `9-mathBot`
* **Concepts applied:** Utilizing the built-in `Math` object to generate random numbers, round decimals, and determine minimum or maximum values.

### 10. `10-fortune-teller`
* **Concepts applied:** Combining random number generation with multiple conditional `if` statements to dynamically select and display a random output.

### 11. `11-build-a-calculator`
* **Concepts applied:** Defining reusable functions with parameters and `return` statements to perform arithmetic operations, while handling edge cases like division by zero using a ternary operator.

### 12. `12-email-masker`
* **Concepts applied:** Combining string manipulation methods like `.indexOf()`, `.slice()`, and `.repeat()` within a function to obscure sensitive email data while preserving the domain.

### 13. `13-loan-qualification-checker`
* **Concepts applied:** Implementing `if...else if` statements and the logical AND (`&&`) operator within a function to evaluate multiple conditions simultaneously.

### 14. `14-celsius-to-fahrenheit-converter`
* **Concepts applied:** Creating a reusable function that uses arithmetic operators to process numerical input, perform mathematical calculations, and `return` a converted value.
